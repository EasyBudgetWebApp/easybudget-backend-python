# EasyBudget Web App.


## Installation..


``` Python
pip install -r requirement.txt
create secret.py with credentials: db_user = '', db_host.., db_pass.., db_port.., db_name..
python manage.py createsuperuser (optional)
python manage.py runserver 0.0.0.0:8000
```
